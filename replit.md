# Vipul Sagar Portfolio

## Overview
A personal portfolio website for Vipul Sagar, an AI/ML Engineer & Full Stack Developer. Built with React, Vite, and Tailwind CSS.

## Project Architecture
- **Framework**: React 19 with Vite (rolldown-vite)
- **Styling**: Tailwind CSS 4 with PostCSS
- **Icons**: lucide-react
- **Project Root**: `vipul-portfolio/` subdirectory

## Key Files
- `vipul-portfolio/vite.config.js` - Vite configuration (port 5000, all hosts allowed)
- `vipul-portfolio/src/App.jsx` - Main application component
- `vipul-portfolio/src/main.jsx` - Entry point
- `vipul-portfolio/index.html` - HTML template
- `vipul-portfolio/tailwind.config.js` - Tailwind configuration
- `vipul-portfolio/postcss.config.js` - PostCSS configuration

## Development
- **Dev server**: `cd vipul-portfolio && npm run dev` (port 5000)
- **Build**: `cd vipul-portfolio && npm run build` (output: `vipul-portfolio/dist/`)

## Deployment
- Static deployment serving `vipul-portfolio/dist/`
